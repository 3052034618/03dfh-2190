## 1. 产品概述

私密约本排期板是一款面向资深车头和工作室主理人的桌面端排期工具，用于高效管理多个熟人圈的剧本杀车局。核心解决"同时开多车、玩家信息碎、靠脑子记不住"的痛点，提供白板式可视化排期、玩家拖拽入位、智能冲突提示和隐私导出能力。

- 目标用户：同时管理 3-10 个熟人车局、经常攒本但不想靠脑子记全部细节的资深车头/工作室主理人
- 核心价值：将分散的约本信息结构化、可视化，减少排期冲突和记忆负担

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 |
|------|------|
| 车头/主理人 | 唯一操作者，管理车局和玩家信息，执行拖拽排期和导出 |

### 2.2 功能模块

1. **排期白板页**：左侧车局列表面板、右侧玩家卡片面板、中央拖拽交互区、智能提示浮层、导出功能
2. **车局管理**：新建/编辑/删除车局，填写本名、人数结构、预计时长、DM、店铺、定金状态
3. **玩家管理**：新建/编辑/删除玩家卡片，填写最近玩过的本、偏好类型、能否熬夜、是否介意反串、常迟到备注
4. **智能提示**：拖拽时自动检测冲突并弹出轻提示
5. **隐私导出**：导出脱敏约本图，仅含必要信息，适合微信群传播

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 排期白板页 | 车局面板(左) | 展示本周所有待开车局卡片，支持新建、编辑、删除；每张车局卡展示本名、人数结构(如3男3女)、预计时长、DM、店铺、定金状态，以及已入位玩家槽位 |
| 排期白板页 | 玩家面板(右) | 展示所有熟人玩家卡片，支持新建、编辑、删除；卡片展示昵称、最近玩过的本(最近3个)、偏好类型标签、熬夜图标、反串图标、迟到备注 |
| 排期白板页 | 拖拽交互 | 从右侧玩家面板拖拽玩家卡片到左侧车局空槽位；拖拽时高亮可放置区域；已入位玩家可拖出移除 |
| 排期白板页 | 智能提示 | 拖拽悬停时检测：1)该玩家上周玩过同店同类型 2)当前车局缺某类玩家 3)两名玩家时间不重合 4)玩家已在本周其他车局中；以轻量toast形式展示 |
| 排期白板页 | 导出约本图 | 点击导出按钮，生成一张包含车局基本信息(本名、人数、时长、DM、店铺、定金)和已入位玩家(仅昵称)的图片，不暴露玩家详细偏好和备注 |

## 3. 核心流程

1. 车头打开排期白板，看到本周待开的车局列表和右侧玩家池
2. 点击"新建车局"，填写本名、人数结构(如3男3女)、预计时长、DM、店铺、定金状态
3. 在右侧玩家面板查看玩家信息，将合适的玩家卡片拖入对应车局槽位
4. 拖拽过程中系统自动检测冲突并弹出轻提示，车头可决定是否忽略
5. 所有车局排好后，点击"导出约本图"，生成脱敏图片发到微信群

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：简洁白板/工位板风格，浅灰白底色，干净利落，专业务实
- **主色调**：冷灰白(#FAFAFA)为底，深墨(#1A1A2E)为文字主色，点缀橙黄(#F59E0B)作为高亮/警告色
- **辅色调**：薄荷绿(#10B981)表示状态正常，淡蓝(#3B82F6)表示信息，玫红(#F43F5E)表示冲突/警告
- **按钮风格**：圆角矩形，扁平设计，hover 微浮起
- **字体**：思源黑体/Noto Sans SC 为主字体，数字使用等宽风格
- **布局风格**：左右分栏白板布局，左侧60%车局面板，右侧40%玩家面板，顶部简洁工具栏
- **图标风格**：线性图标(Lucide)，2px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 排期白板页 | 顶部工具栏 | Logo+标题、本周日期范围、新建车局按钮、导出按钮、设置按钮 |
| 排期白板页 | 车局面板(左) | 卡片列表，每张卡片含本名标题、人数结构标签(男/女图标)、时长、DM名、店铺名、定金状态徽章(已付/未付)、玩家槽位(圆形头像占位)、编辑/删除图标 |
| 排期白板页 | 玩家面板(右) | 顶部搜索/筛选栏，卡片网格布局，每张玩家卡片含昵称、偏好标签(色彩编码)、最近玩过的本(缩略)、熬夜图标🌙、反串图标🎭、迟到备注(红点提示) |
| 排期白板页 | 拖拽态 | 玩家卡片变为半透明跟随光标，车局空槽位高亮为虚线框+薄荷绿底色，已有同局玩家名变灰 |
| 排期白板页 | 智能提示 | 轻量 toast，从拖拽目标附近弹出，带对应图标(冲突⚠️/建议💡/提醒🔔)，3秒自动消失 |
| 排期白板页 | 导出预览 | 模态框展示导出图预览，仅含车局基本信息和玩家昵称，确认后下载PNG |

### 4.3 响应式设计

- 桌面优先设计，目标分辨率 1440x900 及以上
- 最小支持 1280px 宽度，低于此宽度左右面板可折叠
- 不做移动端适配（桌面端专用工具）
